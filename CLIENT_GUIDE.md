# Admin Panel - Quick Start Guide for Clients

Welcome! You can now edit your website content directly from your browser without touching code.

## Accessing the Admin Panel

1. Go to: **https://your-website.com/admin/**
2. Enter your **Admin Password**
3. Click **Login**

## What Happens When I Save?

When you click **"Save"**, the changes are:
1. Sent to the server
2. Stored in the database
3. Immediately reflected on your website

**There's no "publish" step** - your changes go live instantly!

## Logging Out

Click **"Logout"** in the top right to safely exit the admin panel.

For security, you'll need to login again if you want to make more changes.

## Troubleshooting

### "Invalid password" error
- Check that you're entering the password correctly
- Passwords are case-sensitive
- Ask your administrator if you don't remember the password

### Images not showing after I add them
- Make sure you're using a complete URL (starting with http:// or https://)
- Check that the image URL is correct
- The image file might be moved or deleted

### Changes not appearing on the website
- Wait 30 seconds and refresh your browser (Ctrl+R or Cmd+R)
- Clear your browser cache if the change still doesn't show
- Check that you clicked "Save" button

### Can't access the admin panel
- Make sure you're using the correct URL
- Check your internet connection
- Try a different browser
- Contact your developer if the problem continues
